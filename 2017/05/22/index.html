Hugo博客搭建

hugo号称是全球最快的网站构建框架 非常适合刚入门的同学拿来写自己的博客

用来记录自己的学习过程

Hugo博客搭建过程

此处全部是以在windows系统中操作为例

但不管是什么系统，大概的操作都一样的

1.安装Hugo
2.生成站点
3.创建文章
4.安装主题
5.运行hugo

安装Hugo

需要访问github下载
点击下载访问下载链接
如图所示 下载相应系统版本的hugo

生成站点

根据上边提示 你应该已经下载好了hugo的版本 使用代码:
希望生成到 /path/to/site 路径:

hugo new site /path/to/site

这样就在 /path/to/site 目录里生成了初始站点，进去目录:

cd /path/to/site

创建文章

创建一个名称为about的页面：

hugo new about.md

此处推荐使用sublime进行编写 方便快捷精简 此处在content文件夹中生成一个md格式的文档，文档内容采用markdown格式编写
编写如下内容

创建你的第一篇文章，放到post目录，方便之后生成聚合页面

hugo new post/first.md

打开编辑post/first.md

运行hugo

此处可以先不用安装主题 观察生成效果
转到根目录执行hugo命令进行调试

hugo server --theme=hyde --buildDrafts

如果不出什么意外 应该会出现如下图所示内容

打开浏览器 输入 http://localhost:1313/ 就会生成一个hugo的初始页面 之前写的.md中的内容也会出现在页面中

安装主题

我们把安装主题的步骤放在最后的原因是你可以通过网上的一些有关于hugo的文章自己来进行hugo主题的编写
此处我们用一个叫做icarus的主题进行演示
1.cd进入themes目录
icarus主题地址
2.使用git命令行导入

 git clone  https://github.com/digitalcraftsman/hugo-icarus-theme.git

3.转回根目录进行调试
此处我们需要将themes中的config.toml文件覆盖掉根目录中的config.toml
进入到你下载的主题中 直接搜索该文件就可以快速找到 然后右击打开该文件所在位置 一定要在theme中的某个文件夹中
不能是根目录中的config.toml文件

hugo server -t=hugo-icarus-theme --buildDrafts

此处运行成功就会生成目标地址 在网页中输入即可访问

到此我们就算是完成了第一步 在你下载的主题中会有一个reademe.md 我们可以根据这个文档来做修改和完善

一定要认真阅读 看不懂的地方就去翻译 刚开始可能不太好操作 慢慢来 一定要坚持 细心 加油❤
